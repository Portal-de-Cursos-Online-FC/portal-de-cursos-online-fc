1ª Aula - Tutorial HTML

Introdução

Para construir websites, você deve conhecer sobre HTML, a tecnologia fundamental usada para construir a estrutura de uma página web. HTML é usado para especificar se o conteúdo da web deve ser reconhecido como um parágrafo, uma lista, um título, um link, uma imagem, um visualizador multimídia, um formulário ou um entre muitos outros elementos disponíveis ou até mesmo um novo elemento definido por você.

O Curso HTML5

Este curso de HTML5, providencia conceitos básicos e avançados de HTML. O nosso curso é desenvolvido para iniciantes e profissionais. Neste curso todo tópico é dado passo-a-passo para que você possa aprende-lo de uma forma muito fácil.
Se voce é novo no aprendizado de HTML, então voce pode aprender do básico ao nível profissional e depois de aprender HTML com CSS e JavaScript, você será capaz de criar o seu próprio website interactivo e dinámico. Mas agora vamos focar apenas no HTML neste curso.


Os maiores pontos do HTML são apresentados abaixo:


  • HTML, significa HyperText Markup Language. "Linguagem de Marcação de Hypertexto".
  • HTML, é usado para criar páginas e aplicações web.
  • Podemos criar websites estáticos usando apenas HTML.
  • Tecnicamente, HTML é uma linguagem de marcação, pelo que não é uma linguagem de programação.
