6ª Aula - Formatação do HTML

Formatação do HTML


A Formatação do HTML é um processo de formatação de texto para a apresentação do documento. O HTML providencia-nos a abilidade de formatar texto sem usar CSS. Há muitas tags de formatação no HTML. Estas tags são usadas para transformar o texto em negrito, itálico ou sublinhado. Há quase 14 opções disponíveis para manipular a forma que o texto aparece no HTML e XHTML.


No HTML a formatação de tags é dividida em duas categorias:


  • Tag Física: Estas tags são usadas para providenciar a aparência visual do texto.
  • Tag Lógica Estas tags são usadas para adicionar algum valor lógico ou semantico para o texto.

Nota: Há algumas tags físicas e lógicas que podem dar aparência visual, mas elas serão diferentes na semântica.


Aqui, nós vamos aprender 14 tags de formatação do HTML. A seguir apresentamos a lista das tags de formatação de texto.
